# 小红书研究单页导出器

这是一个最小权限、单页、用户触发的 Chrome/Edge 扩展。它不会自动打开链接、自动翻页、处理验证、调用接口或下载图片视频。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`，或 Edge 的 `edge://extensions/`。
2. 打开右上角"开发者模式"。
3. 点击"加载已解压的扩展程序"。
4. 选择本文件夹 `manual_export_extension`。
5. 在扩展菜单中固定"小红书研究单页导出器"。

## 使用

1. 正常登录并手动打开一篇笔记。
2. 等待页面上的正文和互动数据完整显示。
3. 点击扩展按钮。
4. 检查预览，确认后下载 JSON。
5. 手动打开下一篇并重复。

文件保存在浏览器下载目录的 `xhs_research_exports` 子文件夹。

## 数据与安全

- 只在点击按钮时获得当前标签页权限。
- 只读取当前页面可见 DOM；字段无法识别时保存为 `null`。
- 页面出现登录、验证码、安全验证或访问频繁提示时停止导出。
- 保存的网址删除全部查询参数，不包含 `xsec_token`。
- 不采集头像、媒体链接、评论内容、Cookie、LocalStorage 或 User-Agent。
- 源码中不包含 `fetch`、`XMLHttpRequest`、`WebSocket` 或外部脚本。

网页结构变化可能导致字段缺失。每次导出前应查看预览；后续仍需人工核验数据质量。
